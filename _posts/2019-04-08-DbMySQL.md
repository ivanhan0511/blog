## About slave
---
### Fix slave status
- Exec these commands first
  ```shell
  show slave status\G
  stop slave;
  start slave;
  reset slave;
  ```

- Sync DB, only in HotDoor
  ```shell
  /usr/local/gxycloud/db-sync-repair
  ```
